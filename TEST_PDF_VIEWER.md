## 📱 الميزات التقنية

### الصور
- **العدد**: 604 صفحة

### التكامل
- ✓ يكتشف الصور تلقائياً
- ✓ يستخدم عرض PDF إذا كانت الصور موجودة
- ✓ يعود للنصوص إذا لم تكن الصور موجودة
- ✓ يدعم اللغات (العربية والإنجليزية)

### API
- ✓ Tafsir (التفسير) من api.alquran.cloud
- ✓ Translation (الترجمة) من قاعدة البيانات
- ✓ Asbab al-Nuzul (أسباب النزول)

## ⚠️ ملاحظة

الصور الحالية هي **صور بديلة للاختبار** فقط. تحتوي على:
- إطار بسيط
- رقم الصفحة

لاستخدام صور حقيقية من quran.pdf، تحتاج إلى:
1. تثبيت Ghostscript من: https://www.ghostscript.com/download/gsdnld.html
2. تشغيل: `php artisan quran:convert-pdf`
